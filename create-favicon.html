Xen-AI Favicon Generator

Instructions:

  1. Select the desired size from the dropdown
  2. Click "Generate Favicon" to create the logo
  3. Click "Download PNG" to save the file
  4. Generate all required sizes (16x16, 32x32, 64x64, 180x180, 192x192, 512x512)
  5. Use an online converter to create favicon.ico from the 16x16 and 32x32 PNG files
  6. Place all files in the public folder
